# 画面一覧

## 視聴者向け

| URL | ページ名 | アクセス条件 | 説明 |
|---|---|---|---|
| `/login` | ログイン | 誰でも | Googleログイン画面 |
| `/` | トップ（講座一覧） | ログイン必須 | 全公開講座を一覧表示 |
| `/courses/[id]` | 講座詳細 | ログイン＋アクセス権 | セクション一覧・最初の動画 |
| `/courses/[id]/unlock` | パスワード解放 | ログイン必須 | note購入パスワードを入力 |
| `/courses/[id]/watch` | 動画視聴 | ログイン＋アクセス権 | YouTube埋め込み＋カリキュラム |
| `/subscription` | サブスク登録 | ログイン必須 | サブスクパスワードを入力 |

## 管理者向け

| URL | ページ名 | アクセス条件 | 説明 |
|---|---|---|---|
| `/admin` | ダッシュボード | admin権限 | 統計・クイックリンク |
| `/admin/courses` | 講座一覧 | admin権限 | 講座の一覧・削除 |
| `/admin/courses/new` | 講座作成 | admin権限 | 新規講座フォーム |
| `/admin/courses/[id]` | 講座編集 | admin権限 | 基本情報編集＋セクション・動画管理 |
| `/admin/passwords` | パスワード管理 | admin権限 | 発行・リセット（単発/サブスク） |

## API

| エンドポイント | メソッド | 説明 |
|---|---|---|
| `/api/auth/[...nextauth]` | GET/POST | next-auth ハンドラ |
| `/api/courses/unlock` | POST | 単発講座パスワード解放 |
| `/api/subscription/unlock` | POST | サブスクパスワード解放 |
| `/api/admin/courses` | POST | 講座作成 |
| `/api/admin/courses/[id]` | PATCH/DELETE | 講座更新・削除 |
| `/api/admin/courses/[id]/sections` | POST | セクション追加 |
| `/api/admin/sections/[id]` | DELETE | セクション削除 |
| `/api/admin/sections/[id]/videos` | POST | 動画追加 |
| `/api/admin/videos/[id]` | DELETE | 動画削除 |
| `/api/admin/passwords/course` | POST | 単発パスワード発行 |
| `/api/admin/passwords/course/[id]/reset` | POST | 単発パスワードリセット |
| `/api/admin/passwords/subscription` | POST | サブスクパスワード発行 |
| `/api/admin/passwords/subscription/[id]/reset` | POST | サブスクパスワードリセット |

## 画面遷移フロー

### 初回ログイン〜視聴まで（単発購入）
```
未ログイン → /login → Googleログイン
→ / (講座一覧) → 鍵アイコンの講座をクリック
→ /courses/[id]/unlock (パスワード入力)
→ / (成功後リダイレクト) → 解放された講座をクリック
→ /courses/[id] (講座詳細)
→ /courses/[id]/watch?v=[videoId] (視聴)
```

### サブスク登録フロー
```
/ (講座一覧) → サブスク限定バッジの講座
→ /subscription (サブスクパスワード入力)
→ / (リダイレクト、サブスク会員として全講座閲覧可能に)
```
