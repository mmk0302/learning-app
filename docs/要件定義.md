# 要件定義

## 概要

YouTubeの動画埋め込みを使った有料動画講座プラットフォーム。  
noteの購入・サブスクと連携し、パスワード方式でコンテンツを解放する。

---

## ユーザー種別

| 種別 | 説明 | 閲覧できる講座 |
|---|---|---|
| 未ログイン | アカウントなし | トップ含め全ページ閲覧不可 |
| 単発会員 | note記事を単発購入した人 | 購入した講座のみ |
| サブスク会員 | noteのサブスクに加入した人 | 全講座（単発・サブスク両方） |
| 管理者 | 運営者本人 | 全ページ＋管理画面 |

---

## アクセス制御

### ログイン
- Googleアカウントでログイン
- ログインしないとトップページを含む全ページ閲覧不可

### 講座解放フロー（単発購入）
1. noteで記事を購入
2. 記事内に記載されたパスワードをプラットフォームで入力
3. メールアドレスとパスワードが1対1で紐付く（**同じパスワードは別アカウントで使用不可**）
4. 解放された講座はログイン後いつでも視聴可能

### 講座解放フロー（サブスク）
1. noteのサブスク記事内のパスワードを入力
2. アカウントにサブスク会員権が付与される
3. サブスク限定を含む全講座が視聴可能に

### パスワードの1対1紐付けルール
- 1つのパスワード = 1つのメールアドレス（先着）
- 同じパスワードを別のメールアドレスで使おうとするとエラー
- 自分が使用済みのパスワードは再入力で冪等（エラーにならない）

---

## コンテンツ構造

```
講座（Course）
 └── セクション（Section）  ← 第1章、第2章…
      └── 動画（Video）     ← YouTube埋め込み
```

---

## トップページの表示仕様

- 全講座を一覧表示（公開済みのみ）
- サブスク限定講座には「サブスク会員限定」バッジを表示
- 単発購入者がサブスク限定講座をクリック → 「この講座はサブスク会員のみ閲覧可能です」と案内
- 単発購入者が未購入の単発講座をクリック → パスワード入力ページへ

## 技術スタック

| 項目 | 技術 |
|---|---|
| フレームワーク | Next.js（App Router） |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS + カスタムCSS（UTAGEライク） |
| 認証 | next-auth v5（Googleログイン） |
| データベース | Turso（SQLite / libSQL） |
| ORM | Drizzle ORM |
| 動画埋め込み | YouTube IFrame API |
| デプロイ | Vercel（予定） |

- UTAGEライクなUI
